GOparser
========

A Python framework for working with gene ontology (GO) terms and annotations.

If you use GOparser, please cite `Wagner (2015) <http://dx.doi.org/10.1101/018705>`_.

Installation
------------

.. code-block:: bash

	$ pip install goparser
