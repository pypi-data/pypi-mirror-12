MPD-Box

Changelog

0.1.1 (2015-11-14)
------------------

**Initial version**

- Web service to read, access, and edit tags
- Reading tags for Raspberry EXPLORE-NFC
- Logging using RotatingFileHandler
- Config file with ConfigParser
- Run as process, daemon, or service
